# Portfólio 

Site pessoal de portfólio desenvolvido para a entrega avaliativa da 2ª Unidade
da disciplina Introdução à Computação (CESAR School).

## Sobre o projeto

Site estático construído com HTML, CSS e JavaScript, com tema visual
inspirado em jogos retrô / pixel art (fontes "Press Start 2P" e "VT323",
bordas quadradas, sombras "blocadas" e paleta neon).

## Páginas

- `template.html` - página inicial, com apresentação, foto e experiência
- `projetos.html` - lista de projetos desenvolvidos
- `contato.html` - contatos, referências de estudo e inspirações

## Tecnologias

- HTML5
- CSS3
- JavaScript
